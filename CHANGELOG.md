# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-01

### Features

- Initial release of REST-SPEC
- Project scaffolding CLI (`rest-spec-create`)
- Standards setup utilities (`rest-base-setup`, `rest-base-setup-enhanced`)
- Update manager for keeping projects in sync (`rest-spec-update`)
- Error dashboard for monitoring (`rest-spec-errors`)
- API documentation generator (`rest-spec-api-doc`)
- Code generator utilities (`rest-spec-code-gen`)
- Performance profiler (`rest-spec-profiler`)
- Comprehensive standards documentation
- Git Flow branching strategy support
- Conventional commits enforcement
